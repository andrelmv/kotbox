## About the Plugin

**Kotlin Toolbox** is a comprehensive plugin for IntelliJ IDEA that provides an essential set of tools to enhance your Kotlin development experience.

---

## Key Features

<div class="grid cards" markdown>

-   :material-lock-check:{ .lg .middle } **JWT Encoder/Decoder**

    ---

    Encode and decode JWT tokens with support for most industry-standard algorithms.

    [:octicons-arrow-right-24: Learn more](features/jwt-encoder-decoder.md)

-   :material-lightbulb-on:{ .lg .middle } **Kotlin Interpolation Inlay Hints**

    ---

    Visualize computed values of string interpolations and constants directly in your code.

    [:octicons-arrow-right-24: Learn more](features/kotlin-inlay-hints.md)

</div>

## Getting Started

<div class="grid" markdown>

<div markdown>

### :material-download: Quick Install

1. Open IntelliJ IDEA
2. Go to **Settings** → **Plugins** → **Marketplace**
3. Search for "Kotlin Toolbox"
4. Click **Install**
5. The plugin will be installed and the IDE will prompt you to restart. After restarting, the plugin is ready to use!

[View full installation guide](installation.md){ .md-button .md-button--primary }

</div>

<div markdown>

### :material-rocket-launch: Start Using

After installation:

1. Go to **View** → **Tool Windows** → **Kotlin Toolbox**
2. In the **Kotlin Toolbox** window, choose the tool tab you want to use
3. Explore the features!

[Explore features](features/jwt-encoder-decoder.md){ .md-button }

</div>

</div>

Generates a type-safe DSL builder for a Kotlin data class.

Place the caret inside a data class declaration, then invoke this intention via Alt+Enter (or Option+Space on macOS) to generate a builder with @DslMarker support. Nested data classes from the same module are expanded recursively into their own builders.

You can also trigger it from the Generate menu (Alt+Insert) or with Alt+Shift+D.

After invoking, a dialog lets you place the generated code in the same file, a new file, or an existing file.

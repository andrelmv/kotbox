Wraps the selected code block with a Kotlin coroutine builder.

Select any block of code inside a Kotlin file, then invoke this intention via Alt+Enter (or Option+Space on macOS) to choose from the available coroutine builders, or use the direct keyboard shortcuts:

BuilderShortcut
async { }Ctrl+Shift+Alt+Y
coroutineScope { }Ctrl+Shift+Alt+O
launch { }Ctrl+Shift+Alt+J
withContext(Dispatchers.IO) { }Ctrl+Shift+Alt+W
supervisorScope { }Ctrl+Shift+Alt+S
runBlocking { }Ctrl+Shift+Alt+B
withTimeout(timeMillis) { }Ctrl+Shift+Alt+T

The required imports are added automatically.

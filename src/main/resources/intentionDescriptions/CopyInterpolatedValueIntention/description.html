Copies the resolved value of a Kotlin string interpolation to the clipboard.

Place the caret inside a property whose initializer is a string template, then invoke this intention via Alt+Enter (or Option+Space on macOS). All interpolated references are resolved and the resulting string is copied to the clipboard.

Only available when all variable references in the string can be statically evaluated (e.g. const val chains). Unresolvable references disable the intention.

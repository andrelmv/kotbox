Generates a Proto3 message definition from a Kotlin data class.

Place the caret inside a data class declaration, then invoke this intention via Alt+Enter (or Option+Space on macOS) to convert the class and its nested types into a .proto file. Scalar types, collections, maps, enums, and nullable fields are all mapped automatically.

You can also trigger it from the Generate menu (Alt+Insert).

After invoking, a dialog lets you write the output to a new .proto file or preview and copy it to the clipboard.
